# Highlighting Translate 產品指南

## 產品願景
建立一個極簡且直覺的 Chrome 擴充功能，讓使用者在瀏覽外文網頁時，只需透過選取文字即可獲得即時、無縫的翻譯回饋。

## 目標用戶
- 經常瀏覽外文網頁的學習者或研究人員。
- 需要快速理解跨國資訊的工作者。
- 對現有翻譯套件介面感到不滿意的簡潔主義者。
- 尋求選取單字或文章段落後能立即獲得翻譯回饋的語言學習者。

## 核心價值與解決方案
- **即時性與無縫整合**：選取文字後無需跳轉頁面或點擊按鈕，立即顯示翻譯，大幅提升閱讀流暢度。
- **智慧語系偵測**：自動識別來源語言並避免同語系翻譯的干擾，優化使用者體驗。
- **高度客製化**：允許使用者調整翻譯延遲、來源/目標語言，甚至開關功能，以符合個人閱讀習慣。

## 關鍵功能
- **選取即譯 (Highlight to Translate)**：滑鼠選取網頁文字後，自動彈出翻譯結果視窗。
- **多語言支援與偵測**：支援中、英、日、韓等多國語言，具備自動偵測來源語言功能。
- **設定面板 (Popup UI)**：提供直覺的介面來配置語系、延遲時間及功能開關。
- **語音朗讀 (Text-to-Speech)**：支援手動點擊按鈕或自動播放朗讀原文，協助使用者學習正確發音。
- **單字本 (Smart Context Cards)**：收藏翻譯紀錄，以卡片形式呈現並擷取當下的完整例句。支援關鍵字高亮、多國語言介面、檢視、刪除與清空功能，資料儲存於本地以支援大量紀錄。
- **網頁單字自動高亮 (Smart Context Highlight)**：自動在網頁中標記已收藏的單字，並透過懸停 Tooltip 即時複習譯文，強化間隔重複學習。

## 未來擴充計畫
- **豐富翻譯來源**：整合 DeepL、OpenAI 等 API 以提供更高品質的翻譯結果。


## 設計原則
- **極簡主義**：介面乾淨，翻譯框與網頁內容自然融合，減少視覺干擾。
- **直覺互動**：操作簡單直接，無需額外學習即可上手。
