# Agent 工作流 (Agent Workflow)

## 第一階段：探索 (序列執行)

1. **Interviewer (或 Hell Int.)**
   - 目標：提取需求
   - 輸出：`.shared/01-requirements.md`
   - 詢問：標準模式或地獄訪談？

2. **UI Sketcher (UI 草圖師)**
   - 目標：Popup/Options 線框圖
   - 輸出：`.shared/02-wireframes.md`

3. **UX Spec Writer (UX 規格撰寫者)**
   - 目標：擴充功能 UX 規格書
   - 輸出：`.shared/03-ux-specification.md`

## 第二階段：規格化 (平行執行)

4. **Extension Architect (擴充功能架構師 - MV3 + Messaging)**
   - 目標：MV3 架構設計
   - 輸出：`.shared/04-tech-architecture.md`

5. **Mermaid Designer (流程圖設計師)**
   - 目標：訊息流圖 (Message flow diagrams)
   - 輸出：`.shared/05-flow-diagrams.md`

6. **Interactive Designer (互動設計師 - 動畫)**
   - 目標：UI 動畫設計
   - 輸出：`.shared/06-animations.md`

## 第三階段：最終定案 (序列執行)

7. **Planner (規劃師)**
   - 目標：專案開發藍圖 (Roadmap)
   - 輸出：`.shared/07-roadmap.md`

8. **Browser QA (瀏覽器測試員)**
   - 目標：擴充功能測試報告
   - 輸出：`.shared/08-qa-report.md`

## 8 Agent 總覽

| #   | Agent                 | 輸出檔案                | 目的                  |
| --- | --------------------- | ----------------------- | --------------------- |
| 1   | Interviewer (或 Hell) | 01-requirements.md      | 提取需求              |
| 2   | UI Sketcher           | 02-wireframes.md        | Popup/Options UI 設計 |
| 3   | UX Spec Writer        | 03-ux-specification.md  | 擴充功能 UX 文件      |
| 4   | Extension Architect   | 04-tech-architecture.md | MV3 架構設計          |
| 5   | Mermaid Designer      | 05-flow-diagrams.md     | 訊息流圖繪製          |
| 6   | Interactive Designer  | 06-animations.md        | UI 動畫設計           |
| 7   | Planner               | 07-roadmap.md           | 開發藍圖              |
| 8   | Browser QA            | 08-qa-report.md         | 擴充功能測試          |
