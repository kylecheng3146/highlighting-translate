# .shared 資料夾結構 (.shared Folder Structure)

所有 Agent 都將其產出物輸出到 `.shared/` 目錄。

```
.shared/
├── 01-requirements.md         # 待建置內容 (Interviewer)
├── 02-wireframes.md           # Popup/Options 佈局 (UI Sketcher)
├── 03-ux-specification.md     # 擴充功能 UX 模式 (UX Spec Writer)
├── 04-tech-architecture.md    # MV3 + Messaging 架構 (Extension Architect)
├── 05-flow-diagrams.md        # 訊息流圖 (Mermaid Designer)
├── 06-animations.md           # 動畫規格 (Interactive Designer)
├── 07-roadmap.md              # 開發計畫 (Planner)
└── 08-qa-report.md            # 測試結果 (Browser QA)
```

## 最終產出 (Final Outputs)

此工作流的結果將會是 `.shared/` 中的 8 份綜合 Markdown 文件。
