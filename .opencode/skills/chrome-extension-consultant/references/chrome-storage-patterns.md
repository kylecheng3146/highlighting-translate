# 儲存類型 (Storage Types)

| 類型        | 限制  | 同步 | 適用情境                                 |
| ----------- | ----- | ---- | ---------------------------------------- |
| **sync**    | 100KB | 是   | 使用者設定、偏好、小型狀態。跨裝置同步。 |
| **local**   | 5MB   | 否   | 大型數據、快取、離線內容。僅限本機。     |
| **session** | -     | 否   | 暫存數據 (記憶體內)。瀏覽器關閉時清除。  |

## 最佳實踐 (Best Practices)

- 大型數據集優先使用 `chrome.storage.local`。
- 用戶設定嚴格使用 `chrome.storage.sync`。
- 優雅地處理儲存限制。
- 使用 `onChanged` 監聽器來響應更新。
