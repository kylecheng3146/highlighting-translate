# 02. UI Wireframes (UI 草圖)

## 1. 入口 (Entry Point) - Popup

在 Popup 的底部控制列 (或者 History 分頁的上方) 新增一個顯眼的按鈕。

```text
+-----------------------+
|  History List         |
|                       |
|  [Apple] 20%          |
|  [Banana] 100% (Arc)  |
|                       |
|  ...                  |
+-----------------------+
| [START REVIEW QUIZ]   |  <-- Primary Button
+-----------------------+
```

## 2. 測驗頁面 (Quiz Page) - New Tab

全螢幕的新分頁設計，專注於沉浸式體驗。

### 2.1 測驗中 (In Progress)

```text
+--------------------------------------------------+
|  [Exit]                                    3/10  | <--- Progress Indicator
+--------------------------------------------------+
|                                                  |
|                                                  |
|                 頻 果 (中文)                      | <--- Question (Source)
|                                                  |
|                                                  |
+--------------------------------------------------+
|  +----------------+    +----------------+        |
|  |     BANANA     |    |      APPLE     |        | <--- Options (Cards)
|  +----------------+    +----------------+        |
|                                                  |
|  +----------------+    +----------------+        |
|  |     ORANGE     |    |      GRAPE     |        |
|  +----------------+    +----------------+        |
+--------------------------------------------------+
```

**互動狀態**:

- **Hover**: 卡片輕微上浮，陰影加深。
- **Correct**: 被選卡片變綠色，顯示 "Correct!" 動畫 (勾勾)，其餘卡片變淡。
- **Wrong**: 被選卡片變紅，顯示 "Wrong!" 動畫 (叉叉)，**正確卡片同時高亮綠色框**。

### 2.2 結算頁面 (Summary Result) - 10 題結束後

```text
+--------------------------------------------------+
|                                                  |
|                 Review Complete!                 |
|                                                  |
|                 Score: 80%                       |
|                                                  |
|          [Replay]       [Back to List]           |
|                                                  |
+--------------------------------------------------+
```

## 3. 列表頁面更新 (History List Update)

在原有的歷史列表項目中增加進度資訊。

```text
[Existing Item Layout]
+------------------------------------+
|  Apple                             |
|  蘋果                               |
|  [==== 20% ====     ]   [Archive]  | <--- New Elements
+------------------------------------+
```

- **Progress Bar**: 視覺化顯示 `learningRate`。
- **Archive Button**: 點擊後將該項目移出列表 (或變灰/打勾)。
